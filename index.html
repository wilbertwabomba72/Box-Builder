Box Builder

Catch the right toy blocks. Build the pattern. Keep the box balanced.

Box Builder is a cozy offline puzzle game set in a bright toy room. Move the cardboard box across a wooden plank, collect the exact blocks shown in each pattern, and avoid the wrong pieces.

Offline Play 32 Levels Star Ratings Toy Room Style

Build Colorful Toy Patterns

Pattern Goals

Each level shows a playful shape made from blocks, such as houses, flowers, castles, rockets, crowns, and toy vehicles.

Balance Challenge

Move the box left and right on a wooden plank. Catch the correct blocks without letting the box fall.

Progression

Unlock 32 handcrafted levels with increasing block counts, more colors, extra shapes, faster drops, and trickier distractors.

Designed for Quick, Cheerful Play

Box Builder is made for short sessions, gentle challenge, and satisfying pattern completion. It works as a single-player game and stores progress locally on the device.
